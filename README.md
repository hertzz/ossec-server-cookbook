# OSSEC-HIDS Server Cookbook

This cookbook provides a Linux installation and base configuration
for Trend Micro OSSEC-HIDS server.

## Supported Platforms

The following platforms and versions are tested and supported:

  - AmazonLinux 2014.09
  - CentOS 6.5, 7.0
